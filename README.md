# zl_data_view

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test

### 以下依赖如有遗漏，自行安装

## 安装 element-ui ui框架

npm i element-ui -S 

## 安装axios ajax请求

npm  install  --save  axios  vue-axios  qs

## 搭建express -- node.js服务器搭建

npm install express --save

## 安装mongoDb依赖

npm i mongodb mongoose --save

## 安装vuex 

npm install vuex --save

## 安装低版本的less -- 高版本的less可能使用不了

npm install less --save-dev
npm install less-loader@5.0.0 -s

## 安装DataV

npm install @jiaminghi/data-view

## store安装

npm install --save store

## 处理时间戳

npm i moment

## 安装vue-echart

npm install echarts vue-echarts


## 运行run.bat即可，自行设置数据库
